# Домашнее задание L13
## Запуск
Для работы запуска основного кода требуется MySQL база данных, а также свободный порт 8080 на котором будет запущен веб-сервер.
Базу с необходимой схемой, а также standalone jetty сервер можно запустить в виде Docker образов. 
Для этого необходим Docker и Docker Compose. 

Для поднятия контейнера необходимо в консоли выполнить комаду

`docker-compose up`

Для завершения работы нужно выполнить:
 
 `docker-compose down`
 
MySQL поднимается на порту 33306 с базой otus и логином/паролем otus/otus. Web сервер доступн по адресу 8080. 

Папка `deploy` прокидывается в jetty сервер в папку `/var/lib/jetty/webapps/`, поэтому после выполнения `mvn package` 
должен произойти автоматический деплой на сервер (лучше всего, чтобы сервер уже был запущен).
